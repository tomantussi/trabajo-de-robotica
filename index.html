INFORMACION VS DATOS

Datos:es la representacion simbolica de un hecho a traves de un valor que puede ser numerico, una palabra o un simbolo.

Informacion: Es el significado que tiene un conjunto de datos una vez que son interpretados, procesados y comunidados.

Medios de Comunicacion

Conocimiento:Es la mezcla de datos, informacion y experiencia que servia para construir nueva informacion o experiencia que seran utiles para la accion

Transmisión de datos: Tiene varios datos que son:

  • Transmisor: dispositivo u objeto que envia
  • Receptor: dispositivo u objeto que recibe mensajes
  • Canal o medio de comunicacion: es el encargado de transportar el mensaje
